Website — Pengantar & Dasar-Dasar

Konsep umum, jenis, struktur, dan contoh website nyata

Apa itu Website?

Website adalah kumpulan halaman digital yang saling terhubung dan dapat diakses melalui internet menggunakan browser (Chrome, Firefox, Edge, Safari). Setiap website memiliki alamat unik (URL/domain) seperti www.google.com yang menjadi identitasnya di dunia maya.

💡 Analogi Sederhana: Bayangkan internet seperti sebuah kota besar. Setiap website adalah sebuah gedung di kota itu. Domain adalah alamat gedung, dan halaman-halamannya adalah ruangan di dalam gedung. Kamu masuk lewat pintu utama (homepage) dan bisa berpindah ruangan melalui link.

Bagaimana Website Bekerja?

🔒 https://www.contohwebsite.com
Alur saat kamu membuka website:
1
Kamu ketik URL di browser

Misalnya www.tokopedia.com → browser mengirim permintaan ke internet

2
DNS mencari Server

Domain Name System (DNS) menerjemahkan nama domain menjadi alamat IP server yang menyimpan file website

3
Server mengirim file HTML/CSS/JS

Server hosting mengirimkan file-file halaman web ke browser kamu

4
Browser merender tampilan

Browser membaca kode HTML, CSS, dan JavaScript lalu menampilkan halaman visual yang kamu lihat

Tiga Komponen Utama Website

🏗️ HTML — Struktur Seperti kerangka tulang manusia. HTML mendefinisikan elemen: judul, paragraf, gambar, tombol, tabel. Tanpa HTML, tidak ada konten yang tampil.
🎨 CSS — Tampilan Seperti pakaian dan makeup. CSS mengatur warna, ukuran font, jarak antar elemen, dan tata letak halaman agar terlihat indah.
⚡ JavaScript — Interaktivitas Seperti sistem saraf. JavaScript membuat website "hidup": menu dropdown, animasi, validasi form, dan fitur dinamis lainnya.

Jenis-Jenis Website

🏠 Website Profil / Company Profile Menampilkan identitas, sejarah, produk/jasa, dan kontak perusahaan atau organisasi
🛒 Website E-Commerce Platform jual-beli online dengan fitur keranjang, pembayaran, dan manajemen pesanan
📰 Website Berita / Blog Menerbitkan artikel, berita, atau konten secara berkala dengan sistem CMS
📱 Web Application (Web App) Aplikasi berbasis browser: Google Docs, Figma, Canva — fungsionalitas setara aplikasi desktop
🎓 Website Edukasi Platform belajar online: video, kuis, sertifikat, dan manajemen kursus
🎯 Landing Page Satu halaman terfokus untuk kampanye pemasaran atau promosi produk tertentu

Website, Domain, dan Hosting

🏠 Website

Kumpulan file HTML, CSS, JS, gambar yang membentuk halaman dan konten yang dilihat pengguna.

🔤 Domain

Nama alamat website (misalnya namaku.com). Harus didaftarkan dan diperpanjang setiap tahun. Harga mulai Rp 100.000–Rp 500.000/tahun.

💾 Hosting

Server (komputer) yang menyimpan semua file website dan membuatnya bisa diakses online 24 jam. Seperti "tanah" tempat membangun rumah (website).

🔒 SSL / HTTPS

Sertifikat keamanan yang mengenkripsi data antara pengguna dan server. Ditandai dengan ikon gembok 🔒 di browser. Wajib dimiliki agar dipercaya Google.

FAQ & Ringkasan

Apa perbedaan website statis dan website dinamis?
Website statis memiliki konten yang tetap dan tidak berubah kecuali file-nya diedit secara manual oleh developer. Cocok untuk profil dan portofolio. Website dinamis dapat menampilkan konten berbeda-beda tergantung pengguna, waktu, atau data dari database — seperti media sosial atau e-commerce. Website dinamis lebih kompleks namun jauh lebih fleksibel.
Apa itu CMS dan kapan digunakan?
CMS (Content Management System) adalah sistem yang memungkinkan kamu mengelola konten website tanpa perlu menulis kode. Contoh populer: WordPress (blog, company profile), Shopify (e-commerce), Webflow (desain visual). Gunakan CMS jika kamu ingin update konten sendiri tanpa bergantung developer.
Apa saja yang dibutuhkan untuk membuat website?
Minimal tiga hal: (1) Domain — nama alamat website, (2) Hosting — server penyimpan file, dan (3) File website — kode HTML/CSS/JS atau instalasi CMS. Untuk belajar, kamu bisa membuat website gratis di platform seperti GitHub Pages, Netlify, atau Vercel tanpa membeli hosting berbayar.

UX — User Experience
Pengalaman Pengguna

  • Alur pengguna (user flow)
  • Waktu temukan informasi
  • Tingkat kepuasan

💡 Analogi: Sebuah toko fisik — UX adalah denah toko yang logis (produk mudah ditemukan, kasir tidak tersembunyi), sedangkan UI adalah dekorasi toko yang menarik (warna, pencahayaan, rak yang estetis). Toko bisa cantik (UI bagus) tapi pelanggan tetap tersesat (UX buruk) — keduanya harus selaras.

FAQ & Ringkasan

Apa itu wireframe dan mockup? Apa bedanya?
Wireframe adalah sketsa hitam-putih berisi kerangka tata letak tanpa detail visual — seperti denah bangunan. Mockup adalah versi lebih detail dengan warna, font, dan gambar nyata — seperti render 3D bangunan. Wireframe dibuat dulu untuk menyepakati struktur, baru mockup untuk menyepakati tampilan akhir.
Bagaimana cara memilih font yang tepat untuk website?
Gunakan maksimal 2 jenis font: satu untuk judul (display font berkarakter) dan satu untuk isi teks (body font yang mudah dibaca). Hindari font terlalu dekoratif untuk teks panjang. Rekomendasi gratis dari Google Fonts: Poppins, Plus Jakarta Sans, Nunito untuk body — dan Syne, Playfair Display, DM Serif Display untuk judul.
Apa itu User Persona dan mengapa penting?
User Persona adalah profil fiktif yang mewakili tipe pengguna nyata website kamu — meliputi nama, usia, pekerjaan, kebiasaan digital, tujuan, dan frustrasi. Persona membantu desainer membuat keputusan yang berpusat pada pengguna, bukan asumsi pribadi. Misalnya: "Sari, 28 tahun, ibu rumah tangga yang belanja lewat HP" → desain harus mobile-friendly dan checkout cepat.

Apa itu Figma?

Figma adalah aplikasi desain UI/UX berbasis cloud (browser) yang paling banyak digunakan oleh desainer profesional di seluruh dunia. Figma memungkinkan kamu membuat wireframe, mockup, hingga prototipe interaktif yang dapat diklik — tanpa perlu install software apapun, cukup buka browser.

🎉 Gratis untuk Pelajar! Figma memiliki paket gratis (Starter) yang sudah sangat lengkap untuk keperluan belajar dan proyek sekolah. Daftar di figma.com menggunakan akun Google atau email pelajar, dan dapatkan akses fitur Professional secara gratis melalui Figma Education Program.

Alur Pengembangan Desain di Figma

1
Setup Project

Buat File baru → buat Frame dengan ukuran Desktop (1440×900) atau Mobile (375×812) → aktifkan Layout Grid untuk panduan kolom

2
Wireframe

Buat sketsa tata letak menggunakan kotak abu-abu dan placeholder teks. Tentukan posisi navbar, hero section, konten utama, dan footer

3
Style Guide

Definisikan Color Palette, Typography Scale (H1–Body), dan komponen dasar (Button, Card, Input) sebagai design tokens reusable

4
Mockup Visual

Isi wireframe dengan warna nyata, font asli, gambar, dan ikon. Desain setiap komponen dengan detail visual penuh sesuai style guide

5
Component & Auto Layout

Ubah elemen yang digunakan berulang (tombol, navbar, card produk) menjadi Component — edit satu, semua instance ikut berubah

6
Prototype Interaktif

Hubungkan antar frame dengan koneksi interaksi (klik tombol → navigasi ke halaman lain) untuk membuat simulasi website yang dapat diklik

Cara Membuat Clickable Prototype

Prototipe interaktif memungkinkan desain kamu "diklik" seperti website sungguhan, tanpa perlu coding sama sekali:

1
Siapkan Minimal 2 Frame

Contoh: Frame "Beranda" dan Frame "Halaman Produk" yang sudah selesai didesain

2
Klik Tab "Prototype"

Ganti panel kanan dari mode Design ke mode Prototype

3
Pilih Elemen Tombol

Klik tombol/link di canvas → akan muncul titik biru di sisi kanan elemen tersebut

4
Drag ke Frame Tujuan

Tarik titik biru ke frame tujuan → atur: On Click → Navigate To → pilih frame tujuan

5
Pilih Animasi Transisi

Pilih transisi: Instant, Dissolve, Push, atau Slide untuk perpindahan halaman yang halus

6
Preview & Share

Klik ▶ Preview di kanan atas untuk mencoba. Klik Share Prototype untuk kirim link ke teman atau klien

FAQ & Ringkasan

Apa itu Component dan Auto Layout di Figma?
Component adalah elemen desain yang bisa digunakan berulang kali — ubah satu "master component" maka semua instance-nya otomatis ikut berubah. Sangat hemat waktu untuk tombol, navbar, dan card. Auto Layout adalah fitur yang membuat frame menyesuaikan ukuran secara otomatis saat kontennya berubah — seperti Flexbox di CSS. Kedua fitur ini wajib dikuasai untuk desain yang efisien dan scalable.
Bagaimana cara berbagi desain Figma ke orang lain?
Klik tombol Share di pojok kanan atas → atur permission menjadi "Anyone with the link can view" → salin link dan bagikan. Untuk berbagi link prototype yang bisa diklik, masuk ke mode Prototype lalu klik tombol Share Prototype → salin link khusus prototipe yang bisa dibuka siapa saja di browser tanpa akun Figma.
Plugin Figma apa yang berguna untuk desain website?
Plugin rekomendasi: Unsplash — sisipkan foto berkualitas tinggi gratis langsung dari Figma; Iconify — akses ribuan ikon dari berbagai library; Lorem Ipsum — isi otomatis placeholder teks; Contrast — cek rasio kontras warna untuk aksesibilitas; LottieFiles — tambahkan animasi ke desain. Semua bisa diinstal gratis dari menu Plugins di Figma.
